## Dataset: LIAR

Source: https://www.kaggle.com/datasets/doanquanvietnamca/liar-dataset

Description:
The LIAR dataset is a publicly available benchmark dataset for fake news detection.
It contains short political statements labelled across multiple truthfulness categories.

Files:
- train.tsv
- validation.tsv
- test.tsv

Labels:
The dataset contains six labels ranging from false to true.
